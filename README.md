<h1>OpenAI Query with Brave Search</h1>
This project demonstrates how to use the Brave Search API to obtain search results and then feed those results into the OpenAI API to generate a text completion.

<h2>Setup</h2>
Clone this repository.
Create a virtual environment and activate it.
Install the dependencies using pip install -r requirements.txt.
Create a .env file in the root of the project and add your OpenAI and Brave Search API keys like so:

OPENAI_API_KEY=your_openai_api_key
BRAVE_API_KEY=your_brave_api_key

Running the Script
To run the script, use the following command:

python brave.py
